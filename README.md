# Sales-Project
Sales Data Analysis is a data science project focused on analyzing the fourth-quarter sales performance of AAL, a clothing brand operating across multiple states in Australia. The objective of the project is to derive meaningful insights from sales data to help the company make informed decisions for future business expansion.
Sales Data Analysis Project:

Performed data cleaning, normalization, and exploratory data analysis on apparel sales data using Python, Pandas, and NumPy.

Conducted statistical analysis to identify high-performing and low-performing states.

Built visual dashboards using Matplotlib and Seaborn to analyze demographic and time-based sales trends.

Generated data-driven insights to support strategic sales and marketing decisions.
